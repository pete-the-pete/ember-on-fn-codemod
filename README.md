# ember-on-fn-codemod

## Usage

1. Run Codemod
   ```
   npx ember-on-fn-codemod path/of/files/ or/some**/*glob.hbs
   ```

## Contributing

### Installation

- clone the repo
- change into the repo directory
- `yarn`

### Running tests

- `yarn test`

#### Running a single test

- `yarn test --runInBand --testNamePattern=void-elements`
